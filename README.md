# CODTECH IT SOLUTIONS - DATA SCIENCE INTERNSHIP

This repository contains the tasks completed during my Data Science internship at CODTECH IT SOLUTIONS. Each project focuses on a different aspect of the Data Science lifecycle, from ETL pipelines to Deep Learning and Deployment.

---

##  Projects Overview

### 1. Task 1: Data Pipeline Development (ETL)
**Objective:** Create an automated pipeline for data preprocessing, transformation, and loading.
* **Tools:** Pandas, Scikit-Learn
* **Process:** Handles missing values, encodes categorical data, and performs feature scaling (Standardization).
* **Deliverable:** A Python script automating the end-to-end ETL process.

### 2. Task 2: Deep Learning Project (Image Classification)
**Objective:** Implement a Deep Learning model using TensorFlow/Keras.
* **Model:** Convolutional Neural Network (CNN) / Dense Neural Network.
* **Dataset:** MNIST Handwritten Digits.
* **Deliverable:** A functional model with accuracy/loss visualizations to showcase performance.

### 3. Task 3: End-to-End Data Science Project (House Price Prediction)
**Objective:** Develop a full data science project from data collection to web deployment.
* **Algorithm:** Linear Regression
* **Deployment:** Flask Web Framework
* **Deliverable:** A functional web app where users can predict house prices based on square footage.

### 4. Task 4: Optimization Model (Business Profit Maximization)
**Objective:** Solve a business problem using Linear Programming and Optimization techniques.
* **Library:** PuLP
* **Scenario:** Maximizing production profit under labor and material constraints.
* **Insights:** Identifies the optimal product mix to achieve maximum profit (₹2,640).
